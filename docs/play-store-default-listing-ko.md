기준일: 2026-02-25

이 문서는 Google Play Console의 **기본 스토어 등록(Default store listing)**에 입력할 수 있는 초안입니다.

## 1) 앱 세부정보

- 앱 이름 (제안): `AndSSH: 오픈 SSH 터미널`
- 짧은 설명 (80자 이내):
  - `구독·광고 없이 로컬에서 동작하는 오픈소스 Android SSH/SFTP 터미널`

## 2) 자세한 설명 (4000자 이내)

AndSSH는 개발자와 운영자를 위한 오픈소스 Android SSH 터미널 앱입니다.

구독이나 클라우드 의존 없이, 연결 정보와 설정을 기기에 로컬로 저장하고 관리할 수 있습니다. SSH 접속부터 SFTP 파일 작업, 보안 기능, 키 관리, 터미널 커스터마이징까지 실사용에 필요한 기능을 제공합니다.

주요 기능
- SSH-2 접속 (비밀번호, 공개키, 인증서 기반 인증)
- SFTP 파일 브라우저 (업로드, 다운로드, 이름 변경, 권한 변경)
- Mosh 지원 (환경에 따라 SSH 폴백)
- ProxyJump, 포트 포워딩(Local/Remote/Dynamic), Keepalive 설정
- 터미널 스니펫/명령 히스토리, 키 행 커스터마이징, 하드웨어 키 바인딩
- 연결 정보 암호화 저장 (Android Keystore 기반 보호)
- 앱 잠금(비밀번호/생체인증), 자동 잠금, 클립보드 자동 삭제

AndSSH는 다음 사용자를 대상으로 합니다.
- 모바일 환경에서 서버를 직접 운영/관리하는 개발자
- SSH/SFTP 작업을 자주 수행하는 SRE/시스템 관리자
- 광고/구독 없는 로컬 중심 오픈소스 SSH 앱을 원하는 사용자

개인정보 및 데이터 처리
- 개발자 서버로 사용자 데이터를 수집하거나 판매하지 않습니다.
- 앱은 사용자가 설정한 서버에 직접 연결합니다.
- 자세한 내용은 개인정보 처리방침을 참고하세요.

## 2) Full description (max 4000 chars)

AndSSH is an open-source Android SSH terminal app for developers and operators.

It is built around a local-first model: your connection data and app settings are managed on your device, without subscription lock-in or cloud dependence. From SSH access to SFTP file operations, secure key handling, and terminal customization, AndSSH focuses on practical daily workflows.

Key features
- SSH-2 connectivity with password, public key, and certificate-based authentication
- Built-in SFTP browser (upload, download, rename, permission editing)
- Mosh support with SSH fallback when needed
- ProxyJump, local/remote/dynamic port forwarding, and keepalive controls
- Terminal snippets, command history, custom key row, and hardware key bindings
- Encrypted local storage for connection data (Android Keystore-backed protection)
- App lock (password/biometric), auto-lock, and clipboard auto-clear

Who this app is for
- Developers managing servers from mobile devices
- SRE and system administrators who frequently use SSH/SFTP
- Users who prefer an open-source, local-first SSH app without ads or subscriptions

Privacy and data handling
- The developer does not collect or sell personal data through a backend service.
- The app connects directly to servers that you configure.
- See the Privacy Policy for details.
